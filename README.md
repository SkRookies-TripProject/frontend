# 🛫 Trip Project - Frontend

## 📌 프로젝트 소개
여행 정보를 사용자에게 보여주는 웹 프론트엔드입니다.  
백엔드 API와 연동하여 데이터를 시각적으로 제공합니다.

## ⚙️ 기술 스택
- React
- Vite
- Axios
- React Router DOM


## 📂 프로젝트 구조
src
├ api // axios, API 요청 관리
├ assets // 이미지, 아이콘
├ components // 공통 UI 컴포넌트
├ pages // 페이지 단위 컴포넌트
├ hooks // 커스텀 훅
├ utils // 공통 함수
├ styles // 전역 스타일
├ router // 라우팅 설정
├ App.jsx
└ main.jsx

## 🚀 실행 방법
npm install
npm run dev
